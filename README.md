# MusicSDP
MusicSDP is an interface to a music palyer, app built with SwiftUI.

## Features
- Modern and customizable interface using SwiftUI.
- Customizable volume slider.
- Action buttons with musical icons.

## Getting Started

### Prerequisites
- Xcode 26.1.1 or later
- iOS 26.1 or later

### Installation
1. Clone the repository:
```bash
git clone https://github.com/nacher8/MusicSDP26.git
```

2. Open the project in Xcode:
```bash
cd MusicSDP
open MusicSDP.xcodeproj
```

3. Build and run the project on your simulator or device.

## Technologies Used

- **SwiftUI** - Modern declarative UI framework
